# 5.10.2 Adjustment Reason Codes
Template Ref: _mst_01 (Simple Master)

## 5.10.2.1 Business Purpose
Categorizes manual stock adjustments for audit and reporting. Every manual adjustment (5.4.1) must have a reason code for financial audit trails.

## 5.10.2.2 Data Model
| Field | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `code` | String(10) | Yes | e.g. DMG, THF, EXP |
| `name` | String(50) | Yes | Damage, Theft, Expiry |
| `description` | Text | No | |
| `is_active` | Boolean | Yes | |
