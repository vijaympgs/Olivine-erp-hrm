# 5.10.5 Approval Rules
Template Ref: _mst_01 (Simple Master)

## 5.10.5.1 Business Purpose
Rules for determining when a transaction requires manager authorization.

## 5.10.5.2 Data Model
| Field | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `transaction_type` | Enum | Yes | ADJ, TRANSFER, STOCKTAKE |
| `threshold_value` | Decimal | Yes | Above this value requires approval |
| `approval_required` | Boolean | Yes | Global toggle for this type |
| `target_role` | FK (Role) | Yes | Who performs the approval |
