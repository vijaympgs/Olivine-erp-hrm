# 5.10.1 Movement Types
Template Ref: _mst_01 (Simple Master)

## 5.10.1.1 Business Purpose
Defines the classification and behavior of every stock transaction. This is the core "logic engine" that determines if a transaction increases, decreases, or moves stock between locations.

## 5.10.1.2 Data Model
| Field | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `code` | String(10) | Yes | Unique ID (e.g., GRN, ISSUE, ADJ) |
| `name` | String(50) | Yes | Human-readable name |
| `direction` | Enum | Yes | `IN`, `OUT`, `TRANSFER` |
| `is_system` | Boolean | Yes | True for core types (cannot delete) |
