# 5.10.3 Valuation Methods
Template Ref: _mst_01 (Simple Master)

## 5.10.3.1 Business Purpose
Defines the financial logic used to calculate the value of inventory items. This affects cost of goods sold (COGS) and balance sheet reporting.

## 5.10.3.2 Data Model
| Field | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `code` | String(10) | Yes | e.g. FIFO, LIFO, WAC |
| `name` | String(50) | Yes | First In First Out, Weighted Avg Cost |
| `is_active` | Boolean | Yes | |
| `is_default` | Boolean | Yes | Only one default per company |
