# 5.10.4 Inventory Parameters
Template Ref: _mst_01 (Simple Master)

## 5.10.4.1 Business Purpose
Global settings for the inventory module behavior.

## 5.10.4.2 Data Model
| Parameter | Default | Description |
| :--- | :--- | :--- |
| `allow_negative_stock` | False | If True, can ship/issue stock even if system qty < 0 |
| `auto_post_transfers` | False | If True, ships and receives in one click |
| `require_reason_code` | True | Forces reason code on all adjustments |
